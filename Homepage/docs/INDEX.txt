HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

VIEWING THE HOMEPAGE:
1. Double-click index.html to open in default browser
2. Or in terminal: open index.html
3. Or drag index.html into browser window

EDITING THE HOMEPAGE:
1. Open index.html in text editor to change content
2. Open styles.css to change colors or layout
3. Open script.js to modify interactive features
4. Save files and refresh browser to see changes

TESTING RESPONSIVE DESIGN:
1. Press F12 to open browser DevTools
2. Click device toggle (Ctrl+Shift+M)
3. Select different devices to test
4. Test all interactive features

═══════════════════════════════════════════════════════════════════════════════
QUICK CUSTOMIZATION
═══════════════════════════════════════════════════════════════════════════════

CHANGE COLORS:
  Edit styles.css, find :root section, update CSS variables
  Example: --secondary-color: #d32f2f; (change to your color)

UPDATE CONTENT:
  Edit index.html, find the section you want to change
  Example: Change "BMW M440i" to your car model

REPLACE IMAGES:
  Replace emoji placeholders (🚗, 🏎️, etc.) with real images
  Change <span class="card-image">🏎️</span> to
         <img src="path/to/image.jpg" alt="description">

ADD NEW REVIEWS:
  Copy a review-card section and paste it, then edit:
  - Car model name
  - Category
  - Description
  - Rating
