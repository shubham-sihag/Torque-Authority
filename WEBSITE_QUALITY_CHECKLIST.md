## ✍️ **EDITING GUIDE**

### To Update Content
1. Edit HTML files directly in text editor
2. Update text between HTML tags
3. Maintain semantic structure
4. Keep heading hierarchy

### To Change Colors
1. Open `styles-new.css`
2. Find `:root` section (line 16-31)
3. Update color variables
4. Colors auto-update across all pages

### To Add New Images
1. Place images in `Images/` folder
2. Update `src` path in HTML
3. Add descriptive `alt` text
4. Ensure images are JPG format

### To Add New Pages
1. Copy structure from existing review page
2. Update meta tags and title
3. Add content to article-content div
4. Link from homepage review section
5. Add reverse link back to homepage
