## ✏️ How to Edit Content

### To Add Your Articles:
1. Open porsche-review.html, mustang-review.html, or skyline-review.html
2. Find the text that says "Please edit"
3. Replace it with your article content

### To Change Colors:
Edit the `:root` variables in styles-new.css (around line 16-31)

### To Modify Text:
Edit any text directly in the HTML files - no special syntax needed
